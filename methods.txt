
Методы по моему  мнению используются чаще всего:

Списки(list):
    list.append - Добавляет элемент в конец списка
    list.copy() - Копия
    list.remove - Удаление
    list.count - Сохранение
    list.clear - Очистить
Словари(dict):
    dict.get - Возвращает значение
    dict.values - возвращает значения в словаре
    dict.keys - Возвращает ключи
    dict.items - Возвращает пары
    dict.pop -  удаляет ключ и возвращает значение
Множества(set):
    set.intersection(other, ...) или set & other & ... - пересечение
    set.copy() - копия множества
    set.add(elem) - добавляет элемент в множество
    set.discard(elem) - удаляет элемент, если он находится в множестве
    len(s) - число элементов в множестве (размер множества)
    x in s - принадлежит ли x множеству s
Строки(str):
    S.replace - Замена шаблона на замену
    S.split(символ) - 	Разбиение строки по разделителю
    S.isdigit() - состоит ли строка из цифр
    S.upper() - Преобразование строки к верхнему регистру
    S.lower() - Преобразование строки к нижнему регистру
    S.format - Форматированее строк
